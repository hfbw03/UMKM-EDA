# 📊 UMKM Profiling & Segmentation Analysis

Exploratory Data Analysis (EDA) and Business Segmentation of Micro, Small, and Medium Enterprises (UMKM)



## 📌 Project Overview

Usaha Mikro, Kecil, dan Menengah (UMKM) are one of the main drivers of regional economic growth. However, UMKM characteristics vary significantly in terms of assets, workforce composition, production capacity, market reach, and legal status.

This project aims to perform **Exploratory Data Analysis (EDA)** to understand UMKM profiles and develop **rule-based segmentation** to classify businesses based on operational and administrative characteristics. This analysis helps identify patterns, business potential, and segmentation groups that may support data-driven policy making and business development strategies.



## 🎯 Project Objectives

- Understand the general characteristics of UMKM
- Identifying characteristics that tend to be possessed by UMKM with higher turnover performance
- Evaluating the relationship between business legality and business performance
- Segment UMKM into meaningful business groups



## 🔍 Methodology

The project follows a structured data analysis workflow:

### 1. Data Understanding
- Dataset inspection
- Variable identification
- Data type validation
- Missing value detection

### 2. Data Cleaning
- Handling missing values
- Standardizing categorical variables

### 3. Exploratory Data Analysis (EDA)
- Descriptive statistics
- Distribution analysis
- Comparative analysis between UMKM groups

### 4. UMKM Segmentation
Segmentation is conducted using rule-based grouping, which is based on business turnover and legality

## 📈 Key Outputs

The main outputs generated from this project include:

- UMKM profiling based on business characteristics and operational indicators  
- Exploratory Data Analysis (EDA) results describing asset distribution, turnover, workforce composition, etc.
- Rule-based segmentation of UMKM into distinct business groups
